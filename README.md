# SiTech AI School Landing Page

A modern, high-end landing page built with Next.js 15, Tailwind CSS v4, Lucide React, and Framer Motion.

## Features

- **Hero Section**: Animated introduction with glassmorphism effects.
- **Courses Grid**: 4 key courses showcased with hover effects.
- **Features Section**: Highlight of the "Personalized Learning Engine".
- **Responsive Design**: Fully responsive across devices.
- **Dark Mode**: Premium dark theme with vibrant gradients.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Animation**: Framer Motion
- **Deployment**: Vercel
